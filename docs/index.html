Guía de uso para Robot Háptico

Antes de empezar

El laboratorio de control cuenta con dos dispositivos hápticos Touch de 3D Systems. Touch es un dispositivo motorizado que aplica una fuerza de retroalimentación que permite al usuario sentir objetos virtuales y producir sensaciones táctiles a medida que el usuario manipula objetos 3D en pantalla. En esta liga puedes acceder a la guía de usuario del dispositivo.

Para comenzar a utilizar el dispositivo háptico con aplicaciones compatibles es necesario contrar con su respectivo driver. Tan solo basta con descargar el driver más reciente para ser usado con OpenHaptics y correr el instalador. En la siguente liga puedes encontrar la guía de instalación del driver.

Una vez que esté instalado el driver, con la aplicación Touch Smart Setup podrás calibrar el dispositivo antes de usarlo.

Instalación de el API Haptic Touch.

Para iniciar a programar el dispositivo es necesario instalar la librería de Open Haptics. En la siguiente liga puedes encontrar el instalador así como instrucciones para instalar plugins para utilizarlos con Unity o con Unreal.

En esta librería puedes encontrar la documentación completa de las funciones para programar tanto en alto como en bajo nivel. También encontrarás ejemplos de programas ya sea gráficos o en consola.
